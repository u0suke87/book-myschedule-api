#book-myschedule-api document

##まず初めに
このアプリケーションではClient CredentialsのOAuth2.0の認証フローを実装しています。このアプリケーションを使う場合にはGoogle APIsにおいて、OAuthクライアントIDをjson形式で別途作成し、client-secret.jsonとしてこのアプリケーションのホームディレクトリに置いて下さい。
